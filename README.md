# 🚀 TobaKart

TobaKart is a modern local shopping platform that helps users discover products from nearby stores and shop with a fast, simple, and secure experience.

## 🌐 Live Website

https://tobacart.github.io/TobaKart-App/

## ✨ Features

- 🛍️ Modern shopping experience
- 👤 User Panel
- 🛒 Seller Panel
- 🛠️ Admin Panel
- 🚚 Delivery Panel
- 📱 Mobile-friendly UI
- ⚡ Fast loading
- 🔒 Secure authentication
- 🎨 Premium responsive design
